# 健康管理（kenko-kanri）

個人向けダイエット・体組成ダッシュボード。**カロリー収支**（摂取 − 基礎代謝 − NEAT − 運動 − TEF）を中心に、食事・運動・体重・体組成（BMI / LBM / 体脂肪率）を一か所で把握する。iPhone ショートカットで歩数・体重・体組成を同期。LAN / Tailscale 内の個人利用。

**旧名:** 散歩判（sanpo-ban）— v3 でリネーム予定。

## 目的

- カロリー収支の**赤字（マイナス）**を可視化し、ダイエットを継続しやすくする
- 入力負荷を抑えつつ、食事 kcal・PFC・運動・体組成を追える
- Raspberry Pi 上で常時稼働（Tailscale HTTPS 推奨）

## スコープ（最新）

- 含む: Web UI、REST API、MySQL、Health 同期、食事（Myセット・手入力・バーコード）、運動、設定、TOP カード履歴（日/週/月/年）
- 含まない: 認証、散歩セッション記録（v3 で廃止）、週タブ（カード履歴に統合）、PFC 目標・残量表示

## イントーク v3（2026-06-14）— grill-me 合意

### カロリー収支

```
収支 = 摂取 − 基礎代謝 − NEAT − 運動 − TEF
```

| 項 | 定義 |
|----|------|
| 基礎代謝 | Katch–McArdle `370 + 21.6 × LBM(kg)` |
| NEAT | 初期 200 kcal（設定で変更可） |
| TEF | 摂取の 10%（設定で変更可） |
| 運動 | 歩数 kcal + トレッドミル + 筋トレ |
| 符号 | **マイナス = 赤字 = 痩せ方向** |

- LBM / 体組成: **今日 → なければ最新**。一度もなければ基礎代謝カードは同期促し、収支は `--`
- 目標赤字ライン: なし

### TOP 画面

- カード順（スライド通り）: 収支（大）→ 体重 → 摂取 → 基礎代謝 → 消費 → 歩数 → 体脂肪率 → BMI → LBM
- NEAT / TEF はカードなし（収支に含める）
- 各カードタップ → 日 / 週 / 月 / 年

### ナビ

| タブ | 内容 |
|------|------|
| TOP | 収支・体組成カード |
| 食事 | Myセット（旧プリセット）・手入力・バーコード。PFC は**合計 g のみ** |
| 運動 | Myセット・手入力 |
| 設定 | 身長・生年月日・性別・NEAT・TEF 率 |

- **廃止:** 散歩タブ・週タブ・散歩記録（発見メモ含む）
